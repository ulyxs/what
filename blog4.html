Date 26 Oct. 2025

A computer is an electronic device made up of internal and external components that process and store data based on instructions from the user through software programs. It consists of integrated parts that take data as input, process it internally, store the information, and then display the output (GeeksforGreeks, 2025).

I. Hardware Components

Computers are made up of two hardware components: the internal components and the external components. (Ada Computer Science, n.d.)

  • Internal Components – hardware parts essential for processing data and storing information.
  • External Components – hardware components connected to the computer to provide additional features and enhance its overall functionality (Tutorials Point, n.d.; TechTarget, n.d.).
Internal Components External Components
  • Processor (CPU): Executes instructions and processes data (Lenovo, n.d.)
  • Main memory: Stores data and instructions for quick access (GeeksforGeeks, 2025)
  • I/O controllers: Manage communication between the processor and external devices (ITU Online, n.d.)
  • System bus: Connects the main components through the address, data, and control buses
  • Input Devices – allow a user to send instructions and data to the computer
  • Output Devices – show the processed information to the user
  • Storage Devices – long-term storage of processed information

II. Software Components

Software is a set of instructions that, when combined, form programs that tell the computer to perform specific tasks (Tutorials Point, n.d.)

  • Application Software – software designed to carry out specific tasks for the end user.
  • System Software – software written in a low-level language that directly controls the computer’s hardware for efficient performance.
Application Software GENERAL PURPOSE SOFTWARE
Created to handle a broad range of general tasks intended for many users, instead of focusing on one specific purpose.
  • Example: Word processing, spreadsheet, and presentation software
SPECIFIC PURPOSE SOFTWARE
Developed to carry out particular tasks or functions, often tailored for specific users or organizations.
  • Example: Computer-Aided Design (CAD) software, payroll, and resource systems
System Software SYSTEM MANAGEMENT SOFTWARE
The core software that controls a computer’s hardware and software resources, serving as a bridge between the user and the system’s hardware.
  • Example: Operating systems like Windows, macOS, and Android
SYSTEM DEVELOPMENT SOFTWARE
Software used to design, create, test, and maintain other software applications, ensuring their functionality, efficiency, and reliability throughout the development process.
  • Example: Dev-C++ and Microsoft Visual Studio

III. Everyday Application

Computers are now an essential part of our daily lives. In today’s modern era, it is almost impossible to live without owning some form of computer. They are used in various fields and institutions such as medicine, education, government, and business, taking on many different forms and purposes (GeeksforGeeks, 2025).

Education Government Businesses
Computers are used to access information online and serve as platforms for virtual classes and learning management systems Allows citizens to access government services and look up information about the different government information Used to manage data, streamline operations, enhance communication, and support decision-making.

IV. Analysis

Computers have been a part of my life for as long as I can remember. My father, an electrical engineer who loved creating different kinds of circuits and computers, introduced me to them when I was about six years old. I spent my time drawing shapes and painting with colors in Microsoft Paint, which may have sparked my curiosity about computers and how they work. I often wondered, how does the computer detect my clicks? How does it know which shape I want to draw? Those questions became the foundation of my fascination with understanding how a computer works.

To understand computers, one must first know the components that make up the whole system. These include internal parts such as the processor, which acts as the brain of the computer; the main memory, which stores data and information; and the controllers, which manage communication between connected components. Together, these parts allow the computer to receive input, process data, and produce output seamlessly, fulfilling the user’s desired instructions without delay.

However, users may not fully understand how a computer functions at the machine level, which is why software plays an important role. Operating systems help users navigate the many capabilities a computer can offer, such as creating art through programs like Paint. In this way, the operating system acts as a bridge between the hardware and the user, allowing them to communicate and understand each other through application software that receives input and executes instructions. Today, countless applications exist to make our daily lives easier than ever before, and they continue to evolve, helping us solve problems we face each day.

When you have a powerful system like a computer, it inevitably becomes integrated into people’s daily lives. With its endless capabilities and wide range of applications across various fields, the computer has become a necessity. It now serves as a tool that assists and supports us in countless ways, even when we try to resist its influence. In one way or another, our interaction with the world today always involves a computer. Therefore, we must embrace these changes as we continue to move forward into the next stage of technological progress, one that ultimately helps make our lives better and more convenient.
